Problem:
The current parser only recognizes a limited subset of ANTLR v4 syntax.

Missing Features:
1. Quantifiers: *, +, ?, {n,m}, *?, +?
2. Special Sequences: . (any character), Escaped characters
3. Grouping: (...), |
4. Actions and Modifiers: -> operator, fragment keyword, mode declarations
5. Fragment References

Failing Cases:
COMMENT : '<!--' .*? '-->' ;
DTD : '<!' .*? '>' -> skip ;
CharRef : '&#' DIGIT+ ';' ;
Name : NameStartChar NameChar* ;
fragment DIGIT: [0-9] ;
